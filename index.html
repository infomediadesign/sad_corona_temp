Study material

Distributed Systems 3rd edition

The primary literature in this course is the book Distributed Systems 3rd edition (2017) by Maarten van Steen and A.S. Tanenbaum.

In consultation with the author, you can obtain the book free of charge from Maarten van Steen's website.

https://www.distributed-systems.net/index.php/books/ds3/

The book is also available as a printed book via Amazon. 

Pro Git book

The Pro Git book by Scott Chacon and Ben Straub can be read online and downloaded here for free:

https://git-scm.com/book/en/v2

Print versions of the book are available on Amazon.

Excerpts from this book will be made available directly as reading material during the course (content is licensed under the Creative Commons Attribution Non Commercial Share Alike 3.0 license).

GitHub Account

The use of Git is highly recommended for the entire course and all exercises. The final project must also be realized with GIT.

Be sure to commit as many commits to your repository as possible. This not only makes your development process easier and minimizes errors, but also makes it easier to understand how you worked when evaluating your project.

The final project must also be synchronized to GitHub with all commits.

GitHub Invitation Links

Use the invitation link below to create an individual project repository on GitHub. Only projects submitted through this repository can be graded.Please make sure to use the correct link for your group.

To create a repository you need a free account on Github. Please create an account that uses your university address as email address, otherwise your project cannot be associated.

Do not store any personal information on GitHub, including your name and student number. The repositories you create are private by default. You have full administration rights.
