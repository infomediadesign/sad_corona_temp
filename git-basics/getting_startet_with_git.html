Getting startet with Git

Install Git on your Computer

Duration:
10:00

Now, please check if you have Git already installed on your computer. Even if it is already installed, it is probably a good idea to upgrade to the latest version. Below is some information regarding the availability of Git on different operating systems.

Installing Git

Installing on macOS

There are several ways to install Git on a Mac. The easiest is probably to install the Xcode Command Line Tools. You can do this simply by trying to run git from the Terminal the very first time.

git --version

If you don’t have it installed already, it will prompt you to install it.

Installing on Linux

If you want to install the basic Git tools on Linux via a binary installer, you can generally do so through the package management tool that comes with your distribution.

Installing on Windows

There are also a few ways to install Git on Windows. The most official build is available for download on the Git website. Just go to

https://git-scm.com/download/win

and the download will start automatically. Note that this is a project called Git for Windows, which is separate from Git itself.

Another easy way to get Git installed is by installing GitHub Desktop. The installer includes a command line version of Git as well as a GUI. Please note that sometimes there is confusion about Git and GitHub. Git is the tool itself, while GitHub is a provider that offers hosting of central GIT repositories, among other things. GitHub for Desktop is a separate distribution of Git provided by GitHub.
